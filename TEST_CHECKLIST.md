## 📋 测试场景

### 场景1：自由房间完整流程 ⭐⭐⭐
**预期时间**: 5分钟

1. **创建房间**
   - [ ] 从首页点击"创建房间"
   - [ ] 验证房间创建成功
   - [ ] 记录房间号：`______`

2. **添加记录**
   - [ ] 邀请另一个账号加入
   - [ ] 添加5轮对局记录
   - [ ] 点击"📈 走势"查看趋势图
   - [ ] ✅ **关键验证**: 走势图时间顺序从左到右递增

3. **分享验证**
   - [ ] 点击右上角分享
   - [ ] ✅ 验证标题: "来一起打牌吧！房间号：XXXXXX"
   - [ ] ✅ 验证图标: 应用默认图标
   - [ ] ✅ 验证链接不包含 `&cid=`

4. **离开房间**
   - [ ] 点击"离开房间"
   - [ ] ✅ 验证返回到首页

5. **结算验证**
   - [ ] 重新进入房间
   - [ ] 点击"结算积分"
   - [ ] 点击"正式结圈"
   - [ ] ✅ 打开云函数日志查看是否有 "Free room settled"

   - [ ] **重复3次**

   - [ ] **等待 5 分钟**
